# Geek Calendar

A subscribable calendar of geeky in-jokes, puns, and meme-holidays. May the 4th, Pi Day, Towel Day, Mole Day, MAR10 Day, Talk Like a Pirate Day, Hobbit Day, and 40-odd more.

The single source is [`geek-calendar.ics`](geek-calendar.ics) — 64 events, all all-day, most recurring yearly.

## Subscribe (recommended — auto-updates)

Subscribe via URL so new events show up automatically.

```
https://raw.githubusercontent.com/m1ngw/geek-calendar/main/geek-calendar.ics
```

> If `https://` doesn't work, try `webcal://` instead — many calendar apps prefer it for subscriptions.

### iPhone / iPad
Settings → Calendar → Accounts → Add Account → Other → Add Subscribed Calendar → paste URL.

### macOS Calendar
File → New Calendar Subscription → paste URL.

### Google Calendar (web)
Other calendars → `+` → From URL → paste URL.
*Google refreshes subscribed calendars on its own (sometimes slow) schedule.*

### Android
Subscribe via your Google account on the web (above) — it'll sync to the Google Calendar app on your phone.

### Outlook (web)
Add calendar → Subscribe from web → paste URL.

## One-time import (no auto-updates)
Download [`geek-calendar.ics`](geek-calendar.ics) and open it.
